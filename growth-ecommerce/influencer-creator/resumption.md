## Session scope examples

- "Next wave seeding batch (50 creators)"
- "Paid-partnership vetting + contract batch"
- "Whitelisting / paid amplification launch"
- "Monthly creator performance review"
- "Ambassador program tier design"
- "FTC / brand-safety audit"

## Blocker questions

1. **Any active brand-safety concern with a current partner?** If yes, pause the partnership before any new work — extension / cancellation decision first.
2. **Any FTC compliance miss in published content?** Remediate before new partnerships.
3. **Any contract / legal review delay?** If yes, work shifts to vetting + prep, not launch.
4. **Any creator overlap between campaigns?** Same creator running two brands' content same week dilutes both.
5. **Any platform policy change in recent 30 days?** (IG changing ad rules, TikTok creator payout changes) — adjust strategy.

## Ranked work patterns

### If performance declining
Segment diagnosis:
1. Is CAC rising across all tiers or specific tier?
2. Creative fatigue — same creators with same content running too long?
3. Audience saturation — have we covered the creators' best audiences?
4. Competitor activity — new entrant bidding for same creators?
5. Platform algorithm shift — organic reach down across category?

### If scaling up
1. Test adjacent tiers first — micro works, try mid-tier or nano
2. Test adjacent platforms — IG works, try TikTok or YouTube
3. Test adjacent content formats — if static works, test video or live
4. Do not 10× budget without testing new tier/platform/format first

### If onboarding a new agency / tool
Weeks-to-ROI expectations: tool = 30 days, agency = 60 days. Plan accordingly.

### Monthly review cadence
Standard:
1. CAC by tier + platform + creator
2. Top-5 winning creators → promote to ambassador conversation
3. Bottom-20% creators → kill or re-brief
4. Whitelisting performance → expand budget if outperforming branded paid
5. Brand-safety scan → quarterly audit of active creators

## Working protocol additions

- **Never add new creators mid-month** unless emergency — monthly cadence preserves tracking clarity
- **Never amplify creator content without explicit whitelisting rights**
- **Never extend a partnership without reviewing performance data**
- **Always document creator ghosting** — patterns emerge that affect future vetting
- **Always refresh brand-safety screens** quarterly; past-clean doesn't mean currently-clean


## Common drift patterns

- **Celebrity-tier temptation** — one big-name deal eats the budget with poor ROI. Resist unless brand-lift study justifies.
- **Agency dependency** — full-service agencies optimize for agency margin, not creator ROI. Audit attribution.
- **Discount-code stacking** — creators' codes get shared; fake attribution. Limit code use per creator.
- **Usage-rights expiration** — content evergreens silently expire; audit quarterly.
- **Brief creep** — briefs become longer and more restrictive over time; kills creative fit.
