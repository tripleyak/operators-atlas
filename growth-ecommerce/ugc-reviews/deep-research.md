## Use cases

- **Review platform evaluation** — Yotpo / Okendo / Stamped / Loox / Judge.me / Junip / Bazaarvoice
- **UGC platform evaluation** — Yotpo Visual UGC / Olapic / Pixlee TurnTo
- **Competitive review teardown** — peer brands' review programs + performance
- **Review benchmarks** — velocity, rating, response rate by category
- **UGC-in-ads research** — measured performance vs. branded creative
- **SMS review research** — response rates, platforms, best-practice

## Opinionated source priorities

**Prioritize:**

- **Platform-published benchmarks** — Yotpo, Okendo publish category benchmarks annually
- **Bazaarvoice / Power Reviews research** — enterprise-scale data
- **Shopify Plus case studies** — real brand outcomes
- **BigCommerce / Magento case studies** — platform-adjacent data
- **Operator voice on LinkedIn** — real numbers from named brands
- **Academic research on consumer reviews** — behavioral research on trust, star ratings, response impact

**De-prioritize:**

- Platform sales decks (positioning bias)
- Generic "why reviews matter" content
- AI-generated review strategy content

## Question-set specializations

### For review platform evaluation:

1. Feature comparison — reviews, Q&A, photos/videos, UGC, response management, syndication
2. Integration depth — Shopify, Klaviyo, Recharge, Google
3. Pricing — seat + review-volume tiers
4. Migration cost — review-data portability
5. Automation capability — solicitation flows, response routing
6. Scale ceiling — at 10K reviews/month, still performant?
7. UX quality — both customer-facing + admin
8. Customer references at your scale

### For UGC platform evaluation:

Use `deep-research-technical-reference.md` specializations:

1. Rights capture + legal workflow
2. Content moderation + approval
3. Gallery + ad-library management
4. Shopify / ecommerce integration
5. Analytics + performance measurement
6. Creator / customer submission UX
7. Pricing

### For competitive teardown:

1. Review count + rating by product
2. Platform visible
3. Response quality / cadence
4. UGC in ads vs. branded
5. Photo/video rate in reviews
6. Cross-platform presence

### For benchmarks:

1. Review-to-purchase ratio by category
2. Photo/video rate benchmarks
3. Response SLA industry norms
4. Rating distribution (healthy brands)
5. SMS review response rates
