## Use cases

- **Network evaluation** — CJ / Impact / Rakuten / Awin / ShareASale / Pepperjam comparison
- **In-house tool evaluation** — Refersion / Tapfiliate / LeadDyno / Partnerize
- **Creator-commerce platform evaluation** — ShopMy / LTK / MagicLinks / Howl
- **Content affiliate research** — Wirecutter / Business Insider / category review sites
- **Commission structure research** — benchmarks by category + affiliate type
- **Fraud-detection tooling** — platforms + methodologies

## Opinionated source priorities

**Prioritize:**

- **Network public case studies** — CJ, Impact, Rakuten case studies (read critically)
- **Affiliate industry publications** — Affiliate Summit, Partnership Leaders, PI (Performance Innovate)
- **Operator voice on LinkedIn** — affiliate managers sharing real numbers
- **Attribution research** — Haus, Measured, Rockerbox on attribution complications
- **Financial disclosures** — public affiliate networks (Rakuten, CJ parent) publish segment data

**De-prioritize:**

- Network sales content
- Generic "top 10 affiliate marketing tips"
- Agency-published case studies without verifiable results

## Question-set specializations

### For network evaluation:

1. Affiliate base size + category fit
2. Fraud detection capability
3. Tracking technology (cross-device, first-party cookies, server-side)
4. Pricing — override %, platform fees, minimums
5. Reporting + dashboard depth
6. Customer support quality
7. Scale ceiling + exit cost
8. Integration with analytics / attribution

### For in-house tool evaluation:

Use `deep-research-technical-reference.md` specializations:

1. Tracking tech (client-side / server-side / hybrid)
2. Attribution logic customization
3. Commission calculation flexibility
4. Fraud detection
5. Affiliate portal UX
6. Payment automation
7. Integration (Shopify, analytics, email)
8. Scale + performance

### For content affiliate research:

1. Top review sites for your category
2. Editorial calendars + content cadence
3. Monetization models (Amazon Associates vs. direct affiliate, sponsored vs. organic)
4. Relationship-building process (cold outreach vs. agency vs. existing relationships)
5. Typical commission + content-integration expectations
6. Examples of successful brand-publisher partnerships

### For commission benchmarks:

1. Category-level commission norms
2. Tier structures at peer brands
3. Exclusive-code commission uplift
4. Performance bonus structures
5. Coupon vs. content vs. creator commission gaps

### For fraud research:

1. Common fraud patterns (cookie stuffing, click fraud, return fraud)
2. Detection methodologies
3. Industry tools (TrackingDesk, Anura, FraudShield)
4. Red flags in order patterns
5. Remediation procedures
