## Session scope examples

- "New ambassador cohort onboarding"
- "Community health quarterly review"
- "AMA event series planning"
- "Crisis moderation response"
- "Platform migration assessment"
- "Ambassador performance evaluation"

## Blocker questions

1. **Any active moderation issue / brand-safety event?** Resolve first.
2. **Any platform outage or policy change affecting community?**
3. **Any ambassador contract / payment issue pending?**
4. **Any community-member complaint or PR risk surfacing?**
5. **Any upcoming launch that might disrupt community dynamics?**

## Ranked work patterns

### If community health declining
Diagnostic:
1. Posting rate trend — is engagement actually decreasing?
2. New-member activation — are newcomers silencing out?
3. Response rate — questions going unanswered?
4. Top-poster distribution — too concentrated (power users) or too diffuse?
5. Content themes — stale topics? Over-moderation?

### If ambassador performance varies widely
Standard review:
1. Expectations clear at tier entry?
2. Support (content, tools, recognition) adequate?
3. Compensation aligned with expected output?
4. Community recognition of their work?
5. Tier mobility paths (can seed become active?)

### If brand-safety event
Speed matters:
1. Contain — pause related content, pause if needed
2. Communicate — acknowledge, don't hide
3. Correct — fix the underlying issue
4. Document — for future reference and legal
5. Retrospective — how did this slip through?

### Quarterly community review
Standard:
1. Health metrics — posting rate, response rate, growth
2. Top-contributing members + recognition
3. Ambassador tier distribution + performance
4. Content themes — what's resonating, what's not
5. Event calendar review
6. Moderator capacity check

## Working protocol additions

- **Never delete constructive criticism** — transparent community beats curated
- **Never over-moderate to protect brand** — members sense it + leave
- **Never miss ambassador compensation** — trust compounds, breaks permanently
- **Always respond publicly to brand tags** — presence without dominance
- **Always celebrate member contributions** — public recognition is top retention lever

## Verification commands

```
# Community health
{active posting rate, response rate}    # expect: 5-15% posting, 80% response <24h

# Ambassador tier distribution
{tier counts + activity}                 # expect: healthy pyramid

# Moderator capacity
{mod count vs. active members}           # expect: 1:500-1000 active

# Content / event cadence
{events last 30 days, content schedule} # expect: sustained cadence

# Ambassador compensation
{payment + recognition status}           # expect: no outstanding issues
```

## Common drift patterns

- **Broadcast-only degradation** — brand posts but no member engagement follows
- **Power-user concentration** — 5 people drive 80% of posts; mass leaves
- **Ambassador inflation** — everyone gets tier-1; dilutes program
- **Compensation gaps** — late payments or inconsistent recognition breaks trust
- **Moderation burnout** — under-resourced mod team overloads
