> **Inherits from:** `~/.claude/templates/deep-research-competitive-landscape.md`

## Use cases

- **Event portfolio research** — which trade shows / conferences for your category, at what cost, what ROI
- **Webinar platform evaluation** — ON24 / Demio / Livestorm / Goldcast / Zoom Events
- **Customer conference benchmarks** — SaaStr, Dreamforce, HubSpot INBOUND structure and economics
- **Field event vendor evaluation** — experiential agencies, catering, AV, venue platforms
- **Event attribution methodology** — how to measure events in multi-touch models
- **Competitor event strategy teardown** — where do competitors show up, what do they do there

## Opinionated source priorities

**Prioritize:**

- **Event-organizer content** — show directories, attendee profiles, sponsorship packages
- **Bizzabo / Cvent reports** — industry event data
- **Forrester / Gartner event marketing research**
- **Operator voice on LinkedIn + Slack communities** — real cost + ROI from attendees
- **Conference recap blogs from operators** — first-person accounts
- **SaaStr, Lenny's, CMO Coffee Talk** — for B2B event strategy

**De-prioritize:**

- Event-organizer sales decks (motivated claims)
- Generic "ROI of trade shows" content
- AI-generated event tips

## Question-set specializations

### For event portfolio research:

1. Tier-1 / tier-2 / tier-3 events in your category by ICP-fit
2. Typical attendee profile (title, company size, industry)
3. Sponsorship package costs (booth, speaking slot, hosted event)
4. Historical attendance trend
5. Parallel events / side events at the conference
6. Past brand presence (who sponsored, what worked)
7. ROI benchmarks from attendee reports

### For webinar platform:

1. Registration / attendance UX
2. Multi-session management
3. Content hub / on-demand library
4. Integration (CRM, MAP, sales engagement)
5. Analytics depth
6. Scale ceiling (simultaneous attendees, number of events/month)
7. Pricing
8. Customer quality (operator brand names using it)

### For competitor event strategy teardown:

1. Which events they attend / sponsor
2. Booth size + presence
3. Speaking slots + topics
4. Hosted side events
5. Social promotion patterns
6. Thought-leadership approach

### For attribution methodology:

1. Multi-touch weighting for events (first-touch, last-touch, weighted)
2. Offline-to-online attribution technique
3. Event-specific vs. generic multi-touch credit
4. Pipeline-acceleration measurement
5. Brand-lift measurement adjacent to event exposure
