## Session scope examples

- "Pre-show meeting scheduling push"
- "Booth design review"
- "Post-event follow-up cadence batch"
- "Webinar topic + speaker planning for Q3"
- "Trade show portfolio rationalization"
- "Event attribution methodology update"

## Blocker questions

1. **Any event < 30 days out?** Lock execution; no new planning
2. **Any sponsorship contract pending signature?** Financial + timing risk
3. **Any speaker / SME availability gap?** Confirm before committing to agenda
4. **Any venue / logistics issue** (AV, F&B, space)?
5. **Any post-event follow-up delay?** Every day past Day 7 cuts response rate materially

## Ranked work patterns

### If event is 0-30 days out
Execution mode:
1. Daily check on logistics + content + attendees
2. Speaker / SME prep
3. Booth / demo readiness
4. Pre-show outreach intensity
5. Staff / agency coordination

### If event just completed
Time-sensitive follow-up:
1. Day 1 personalized emails (manual if needed)
2. Day 3 content asset distribution
3. Week 1 retrospective scheduling
4. Pipeline attribution tagging in CRM
5. Content re-use planning

### If portfolio decisions pending
Analysis:
1. Prior-year ROI by event (pipeline + brand + retention)
2. Competitive + strategic fit
3. Budget envelope + constraints
4. Team capacity
5. Decision: attend / skip / expand

### Webinar cadence
Quarterly planning:
1. Topic pipeline — what's timely, what's evergreen?
2. Speaker pipeline — internal + external
3. Promotion plan — email, social, paid
4. Content re-use — clip + repurpose per webinar
5. Performance review — last quarter's benchmarks

## Working protocol additions

- **Never delay post-event follow-up past Day 7** — response rate degrades materially
- **Never commit to speaker without content prep time**
- **Never skip booth staffing plan** — unstaffed moments = missed leads
- **Always capture multi-media** — photos, video, quotes — for content re-use
- **Always document logistics learnings** — small issues compound year-over-year

## Common drift patterns

- **During-event over-investment** — booth glamour + party spend eats the budget; pre + post underfunded
- **Generic follow-up** — templated emails instead of personalized; 5-10× response loss
- **Swag theater** — expensive swag attracts swag-hunters not buyers
- **Trade show accumulation** — adding shows without cutting; budget bloats
- **No retrospective discipline** — next year repeats this year's mistakes
